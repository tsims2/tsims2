## Featured work

### ESP.App — Internal operations application

ESP.App is an internal operations application for managing energy-program workflows across enrollments, events, performance calculations, settlements, reporting, scheduling, run tracking, and artifact generation.

The app connects React and TypeScript frontend workflows with FastAPI backend services, Celery/Redis execution, PostgreSQL app state, Azure SQL operational data, and reusable Python modules from ESP.Artifacts.

Key areas:

* workflow execution
* scheduled jobs
* run history and logs
* performance calculation workflows
* report generation
* admin controls
* feedback loops
* reusable Python domain logic

### ESP.Artifacts — Python domain library

ESP.Artifacts is a reusable Python domain library for energy operations workflows. It centralizes business logic for database access, external data clients, baseline and performance calculations, settlement support, data quality checks, report generation, payment file generation, statement generation, and program-specific automation.
